# Fuentes Legislativas — Federal

## Fuente 1 — Diario Oficial de la Federación (DOF)

- **URL**: https://www.dof.gob.mx/
- **URL índice diario**: https://www.dof.gob.mx/fecha.php?fecha=DD/MM/YYYY
- **Frecuencia**: Diaria (lunes a viernes, y días extraordinarios)
- **Formato**: PDF y HTML
- **Notas**: Publicaciones clasificadas en: PRIMERA, SEGUNDA y TERCERA sección. Buscar en PRIMERA sección para actos legislativos.

## Fuente 2 — Cámara de Diputados (Leyes Federales Vigentes)

- **URL**: https://www.diputados.gob.mx/LeyesBiblio/
- **Estructura**: Lista alfabética + buscador. PDFs y DOC descargables directamente.
- **Frecuencia de actualización**: Al día siguiente de publicación en DOF
- **Formato**: PDF y DOC
- **Notas**: Es la fuente más confiable y completa para leyes federales vigentes. Incluye fecha de última reforma.

## Fuente 3 — SISTA / SCJN

- **URL**: https://legislacion.scjn.gob.mx/
- **Filtro federal**: https://legislacion.scjn.gob.mx/Buscador/Paginas/wfOrdenamiento.aspx?q=
- **Frecuencia**: Variable
- **Formato**: HTML con descarga PDF
- **Notas**: Útil para cruce de información y validación. Incluye tesis y jurisprudencia relacionada.

## Palabras clave de vigilancia DOF

```
Ley, Decreto, Reforma, Reglamento, Abrogación, Derogación, Código, Acuerdo, Norma Oficial Mexicana
```
